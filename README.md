Features:

- Support
  - Transparency 
  - Extended
  - Only 1098, it is not recommended for other versions.

- Image Enhancer
  - Upscale: 2, 4, 8
  - Denoise Level: 1, 2, 3 
  - Resize: 32x32, 64x64, 128x128, 512x512
  - Custom Size
  - Image Edit
    - Mirror Vertical
    - Mirror Horizontal
    - Color Adjust
    - Rotation
 
- Spr/Dat Editor (EXPERIMENTAL)
  - Edit Item/Outfit/Missile/Effects
  - Flags Adjust
  - Mass Edit/Delete
  - Slicer (Obd Style)
  - Sprite Optmizer
  - Import
  - Export
  
- OTB Reload (Dont Work)
  - Flags Reload/Sync
